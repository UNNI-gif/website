# The Professional Couriers - Kecheri

A fast, lightweight, and fully responsive single-page business website built for the Kecheri branch office.

## 🚀 Features
* **Modern UI Header:** Clean navigation featuring an enlarged, high-resolution logo and quick-action contact options.
* **Service Modals:** Interactive popup boxes detailing Domestic, International, Cargo, and Premium courier services.
* **Universal Google Map:** Embedded with precise geographic coordinates (`10°37'09.8"N 76°07'23.7"E`) to load instantly on any user's system.
* **Interactive Contact Dock:** A custom CSS expandable floating action button that opens up a vertical stack linking to WhatsApp, Direct Phone, Email, and Instagram.

## 🛠️ Tech Stack
* HTML5 (Semantic Structure)
* CSS3 (Custom Grid, Flexbox layouts, & smooth transitions)
* JavaScript (Interactive modal state management)
* FontAwesome (Vector iconography)
